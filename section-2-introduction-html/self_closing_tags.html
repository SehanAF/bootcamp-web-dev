Self-closing tags are tags that do not have a closing tag.

  • The <br> tag is a self-closing tag that creates a line break.
  • The <hr> tag is a self-closing tag that creates a horizontal rule.
  • The <img> tag is a self-closing tag that creates an image.

Example

The following code uses self-closing tags to create a simple web page:

```html Self-Closing Tags Example

Self-Closing Tags Example

This is a paragraph.


This is another paragraph.

This is a third paragraph.

``` Ajari saya tentang Self Closing Tags di HTML, berikan contoh nya serta Penjelasan yang mudah dimengerti, berikan yang terbaik.
